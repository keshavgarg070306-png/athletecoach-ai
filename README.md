# ⚡ AthleteCoach AI

A multi-sport AI-powered training platform that generates personalized weekly drill schedules based on player weakness profiles.

## 🌐 Live Demo
- **Frontend:** https://athletecoach-frontend.vercel.app
- **Backend API:** https://your-railway-url.up.railway.app
- **Swagger UI:** https://your-railway-url.up.railway.app/swagger-ui/index.html

## 🛠 Tech Stack

**Backend**
- Spring Boot 3.5
- Spring Security + JWT Authentication
- MySQL + Spring Data JPA
- WebSocket (STOMP)
- Swagger / OpenAPI

**Frontend**
- React + Vite
- Recharts (Analytics)
- Canvas Confetti (Animations)
- CSS Animations

**Deployment**
- Backend → Railway
- Frontend → Vercel
- Database → Railway MySQL

## ✨ Features

- 🎯 Rule-based coaching engine
- 📋 Personalized weekly training plans
- ⚡ XP + Streaks + Badges gamification
- 🏆 Global leaderboard
- 📊 Performance analytics charts
- 🎮 FIFA-style skill ratings
- 🔔 Real-time WebSocket notifications
- 📅 Match fixture management
- 🌙 Dark theme UI

## 🚀 Getting Started

### Backend
```bash
cd athletecoach-ai
./mvnw spring-boot:run
```

### Frontend
```bash
cd athletecoach-frontend
npm install
npm run dev
```
